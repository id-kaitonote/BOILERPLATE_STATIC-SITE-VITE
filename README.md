# Static-Site-Boilerplate-Vite
Web制作における静的サイトを構築するためのボイラープレートです。
主に下記の技術を採用して構築されています。

・Vite
・Pug
・DartSass
・PostCSS
・TypeScript
・Prettier
・stylelint
・ESLint

## Setup

``` bash
yarn
yarn dev
```

## Scripts

``` bash
# 開発用コマンド
yarn run dev
# ビルド用コマンド
yarn run build
# プレビュー用コマンド
yarn run build
```

### Lint & Format

``` bash
# tslint
yarn run lint:ts
# tsformat
yarn run lint:ts
# stylelint
yarn run lint:scss
# styleFormat
yarn run fix:css
```
